#  Day 3: Even or Odd Checker

## 📚 Concepts Covered
- if-else statements
- Modulus operator (%)
- Conditional logic

## 📝Java Program

 public class EvenOdd {
    public static void main(String[] args) {

  int number = 17;

  if (number % 2 == 0) {
            System.out.println(number + " is Even");
        } else {
            System.out.println(number + " is Odd");
        }
    }
}


 The modulus (%) operator gives the remainder after division. It is commonly used to determine whether a number is even or odd.
 Output: 17 is Odd
